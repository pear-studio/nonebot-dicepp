## 指令总览

DicePP 共包含 **29** 个注册用户指令，分布在 9 个功能模块中。

| 模块 | 指令数量 | 主要功能 |
|------|----------|----------|
| roll | 5 | 掷骰、随机选择、骰池、默认骰、业力骰子 |
| character | 2 | 角色卡、生命值管理 |
| common | 10 | 帮助、昵称、日志、日志统计、激活、配置、模式、欢迎词、自定义对话、Master |
| query | 2 | 资料查询、私设管理 |
| deck | 2 | 抽卡、随机生成 |
| initiative | 3 | 先攻列表、投掷先攻、战斗轮 |
| misc | 4 | 今日人品、属性生成、统计、邀请链接处理 |
| dice_hub | 1 | 多机器人互联 |

## 骰子系统 (roll)

骰子系统是 DicePP 的 core 功能，提供丰富的掷骰表达式支持。

### 掷骰指令

**指令名称**: 掷骰指令  
**触发词**: `.r` / `.roll`  
**优先级**: 0  
**Flag**: `DPP_COMMAND_FLAG_ROLL`

**语法格式**:
```
.r[修饰符][表达式] [原因]
```

**修饰符说明**:
| 修饰符 | 说明 |
|--------|------|
| `h` | 暗骰，结果私聊发送 |
| `s` | 缩略模式，只显示最终结果 |
| `a` | 概率检定模式 (COC风格 D100) |
| `n` | 后日谈检定模式 (D10) |
| `exp` | 计算掷骰期望值 |

**表达式语法**:
- 基础骰子: `d4`, `d6`, `d8`, `d10`, `d12`, `d20`, `d100`
- 多骰: `3d6` (掷3个6面骰)
- 加减: `d20+5`, `2d6-2`
- 优劣势: `d20优势`, `d20劣势`
- 取高/低: `4d6k3` (取最高3个), `2d20kl1` (取最低1个)
- 多轮: `3#d20+5` (掷3次)
- 重投: `d20r1` (结果为1时重投)
- 成功计数: `10d10cs>=8` (统计>=8的骰子数)

**示例**:
```
.rd20+1d4+4           # 基础掷骰
.r4#d                  # 投4次d20
.rd20劣势+4            # 带劣势攻击
.r2#d优势+4 攻击地精   # 两次优势攻击并注明原因
.r1d12+2d8+5抗性       # 减半向下取整
.rh d20+5              # 暗骰
.rs 3d6                # 缩略显示
.rexp d20+5            # 计算期望值
```

**配置项**:
| 配置键 | 默认值 | 说明 |
|--------|--------|------|
| `roll_enable` | 1 | 掷骰指令开关 |
| `roll_hide_enable` | 1 | 暗骰指令开关 |
| `roll_exp_cost` | 10 | 计算期望消耗的点数 |

### 随机选择指令

**指令名称**: 随机选择指令  
**触发词**: `.c` / `.choose`  
**优先级**: 0  
**Flag**: `DPP_COMMAND_FLAG_ROLL`

**语法格式**:
```
.c [次数] [选项1] [选项2] [选项3] ...
.c [选项1]/[选项2]/[选项3]
```

**参数说明**:
| 参数 | 必填 | 说明 |
|------|------|------|
| 次数 | 否 | 抽取数量，默认为1 |
| 选项 | 是 | 用空格或 `/` 分隔 |

**示例**:
```
.c 苹果 香蕉 橘子       # 从三个选项中选一个
.c 苹果/香蕉/橘子       # 使用斜杠分隔
.c 2 红 绿 蓝 黄        # 从四个选项中选两个
```

### 业力骰子指令

**指令名称**: 业力骰子指令  
**触发词**: `.karmadice` / `.业力骰子` / `.骰子模式` / `.业力引擎`  
**优先级**: 默认  
**Flag**: `DPP_COMMAND_FLAG_MANAGE`  
**仅群聊**: 是

**语法格式**:
```
.karmadice [子指令] [参数]
```

**子指令说明**:
| 子指令 | 参数 | 说明 |
|--------|------|------|
| `on` | 无 | 启用业力骰子 (需管理员) |
| `off` | 无 | 关闭业力骰子 (需管理员) |
| `status` | 无 | 查看当前配置和统计 |
| `set` | `[期望%] [窗口]` | 配置自定义参数 |
| `mode` | `[模式名]` | 切换预设模式 |
| `engine` | `[引擎名]` | 切换核心算法 |
| `reset` | `[me]` | 清空历史记录 |
| `help` | 无 | 显示帮助 |

**可用模式**:
| 模式 | 说明 |
|------|------|
| `balanced` | 默认，目标55%，适度平衡运气 |
| `hero` | 目标65%，支持绝境爆发 |
| `grim` | 目标40%，乐极生悲 |
| `dramatic` | 加重极端值 |
| `stable` | 三次骰平均，降低波动 |
| `custom` | 手动设置期望与窗口 |

**可用引擎**:
| 引擎 | 说明 |
|------|------|
| `advantage` | 优势判定 |
| `precise` | 精确加权 |

**示例**:
```
.karmadice on                   # 开启业力骰子
.karmadice status               # 查看状态
.karmadice set 60 20            # 设置目标60%，窗口20次
.karmadice mode hero            # 切换到英雄模式
.karmadice reset me             # 清空个人历史
```

**优先级**: 0  
**Flag**: `DPP_COMMAND_FLAG_MANAGE`

```
.nn [昵称]
```

| `first` / `提前` | `[名称]` | 提前某条目 |
| `swap` / `交换` | `[名称1] [名称2]` | 交换两条目位置 |
| `import` / `导入` | `[数据]` | 导入先攻数据 |

**示例**:
```
.init                   # 查看先攻列表
.先攻清除               # 清空先攻列表
.先攻删除地精           # 删除地精
.init del 地精a/地精b   # 删除多个
.先攻提前 骑士          # 将骑士提前
.先攻交换 战士 法师     # 交换战士和法师位置
```

## 开发者指南

### 添加新指令

1. 在 `module/` 目录下创建新模块或在现有模块中添加
2. 继承 `UserCommandBase` 类
3. 使用 `@custom_user_command` 装饰器注册

**基本模板**:

```python
from core.command import UserCommandBase, custom_user_command
from core.command.const import *

@custom_user_command(
    readable_name="指令名称",
    priority=DPP_COMMAND_PRIORITY_DEFAULT,
    group_only=False,
    flag=DPP_COMMAND_FLAG_ROLL
)
class MyCommand(UserCommandBase):
    def __init__(self, bot: Bot):
        super().__init__(bot)
        # 注册本地化文本
        # 注册配置项

    async def can_process_msg(self, msg_str: str, meta: MessageMetaData) -> Tuple[bool, bool, Any]:
        should_proc = msg_str.startswith(".mycmd")
        return should_proc, False, None

    async def process_msg(self, msg_str: str, meta: MessageMetaData, hint: Any) -> List[BotCommandBase]:
        # 处理逻辑
        return [BotSendMsgCommand(...)]

    def get_help(self, keyword: str, meta: MessageMetaData) -> str:
        if keyword == "mycmd":
            return "帮助文本"
        return ""

    def get_description(self) -> str:
        return ".mycmd 指令描述"
```

### 优先级说明

| 常量 | 值 | 说明 |
|------|-----|------|
| `DPP_COMMAND_PRIORITY_MASTER` | -100 | Master级指令 |
| `DPP_COMMAND_PRIORITY_DEFAULT` | 0 | 默认优先级 |
| `DPP_COMMAND_PRIORITY_TRIVIAL` | 100 | 低优先级 |

- 数值越小，优先级越高
- 相同触发词前缀的指令需要注意优先级设置

### Flag 常量

| Flag | 说明 |
|------|------|
| `DPP_COMMAND_FLAG_ROLL` | 掷骰相关 |
| `DPP_COMMAND_FLAG_DND` | DND相关 |
| `DPP_COMMAND_FLAG_COC` | COC相关 |
| `DPP_COMMAND_FLAG_MANAGE` | 管理功能 |
| `DPP_COMMAND_FLAG_HELP` | 帮助功能 |
| `DPP_COMMAND_FLAG_BATTLE` | 战斗相关 |
| `DPP_COMMAND_FLAG_INFO` | 信息查询 |
