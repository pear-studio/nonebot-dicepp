# Roll 模块测试覆盖分析

## 现有测试盘点

### test_roll.py
- **算术测试**: 基本覆盖 (+, -, *, /)，含结合性测试 (1-1-1, 5/2+3/2 等)
- **优先级测试**: 部分覆盖 (1+2*2, 1*2+2)
- **括号测试**: 基本覆盖，含嵌套括号
- **XDY 格式**: 覆盖基础骰子表达式
- **修饰器测试**: K/KH/KL, R/X/XO, CS 有调用但无确定性断言
- **错误路径**: 部分覆盖（边界常量、格式错误、括号不完整）

### test_modifier.py
- **修饰器解析**: 覆盖所有修饰器初始化
- **修饰器执行**: 覆盖 modify() 方法基本路径
- **缺失**: 比较条件 (<=, >=, ==) 的详细测试

### test_result.py
- **RollResult 属性**: 全面覆盖
- **输出格式**: 覆盖 get_result(), get_complete_result() 等

## 覆盖缺口识别

### 1. 算术表达式缺口
- [ ] **优先级边界**: 连续乘除混合 (2*3/4*5)
- [ ] **一元运算符**: 负号表达式 (-D20, --1)
- [ ] **除零行为**: 未测试 (如 D20/0)
- [ ] **确定性断言**: 大多数测试无固定随机源，结果不可重复

### 2. 修饰器缺口
- [ ] **比较条件组合**: CS<=, CS>=, CS== 未单独测试
- [ ] **重骰条件组合**: R<=, R>=, R== 未单独测试
- [ ] **修饰器链**: 多修饰器组合 (如 4D20K2R<5) 无专项测试
- [ ] **边界爆炸**: X 修饰器爆炸次数上限未测试
- [ ] **期望值计算**: expectation() 方法测试缺失

### 3. 错误路径缺口
- [ ] **语法错误归类**: 无统一错误类型断言
- [ ] **运行时错误**: 修饰器对非骰子表达式的应用
- [ ] **限制触发**: 表达式长度、解析深度上限

### 4. 输出格式缺口
- [ ] **过程文本**: 骰子结果样式 ([val] 格式) 无回归断言
- [ ] **修饰器过程**: 如重骰符号 (→) 无断言

## 建议补充

1. **兼容语料库**: 构建固定种子的确定性测试用例
2. **错误类型断言**: 为每类错误增加 error_type 检查
3. **新旧引擎对照**: 创建双引擎并行执行框架
4. **高风险用例标记**: 识别输出格式敏感的表达式
