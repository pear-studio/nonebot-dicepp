## 1. 骰子系统

| 功能 | 命令 | 说明 |
|------|------|------|
| 基础掷骰 | `.r` / `.roll` | 支持多种骰子表达式，如 `3d6+5`、`4d6k3` 等 |
| 随机选择 | `.choose` / `.pick` | 从选项中随机选择，如 `.choose A B C` |
| 骰池 | `.pool` | 骰池投掷，用于 DND 5E 等规则 |
| 默认骰设置 | `.set` | 设置默认骰子面数 |
| 业力骰子 | `.karma` | Karma 系统，记录玩家运气值 |

### 骰子表达式支持

- 基础骰子：`d4`, `d6`, `d8`, `d10`, `d12`, `d20`, `d100`
- 掷骰数量：`3d6` 表示掷 3 个 6 面骰
- 加减修正：`+5`, `-2`
- 优势/劣势：`2d20kh1` (优势), `2d20kl1` (劣势)
- 掷骰原因：`3d6+5 力量检定`

## 10. 外部 API

机器人提供 FastAPI 接口供外部调用：

- `/api/roll` - 掷骰接口
- `/api/query` - 查询接口
- `/api/character` - 角色卡接口

详见 [FastAPI 模块](./fastapi.md)

## 11. 多机器人互联

通过网站 API 实现机器人注册与发现，不再依赖 QQ 消息。

| 功能 | 命令 | 说明 |
|------|------|------|
| 设置 API 地址 | `.hub url <地址>` | 设置 DiceHub 网站 API 地址 |
| 注册机器人 | `.hub register` | 向网站注册机器人，获取 API Key |
| 查看 API Key | `.hub key` | 查看当前机器人的 API Key |
| 查看在线列表 | `.hub list` | 查看在线的机器人列表 |
| 手动心跳 | `.hub online` | 手动触发一次心跳 |
| 查看状态 | `.hub status` | 查看机器人注册状态 |

## 附录

### 数据存储

机器人使用 JSON 格式存储以下数据：

- 用户数据 (`DC_USER_DATA`)
- 群组数据 (`DC_GROUP_DATA`)
- 昵称 (`DC_NICKNAME`)
- 宏定义 (`DC_MACRO`)
- 变量 (`DC_VARIABLE`)
- 角色卡 (`DC_CHAR_DND`)
- 生命值 (`DC_CHAR_HP`)

### 国际化

支持多语言文本，可通过 `LocalizationManager` 管理。
